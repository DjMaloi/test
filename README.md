# SupportAI Bot — Telegram-бот с ИИ и базой знаний

## Быстрый старт
1. Установи зависимости: pip install -r requirements.txt
2. Заполни .env (токен бота + Groq ключ)
3. Запусти: python bot.py

## Ключи
- Bot: @BotFather → /newbot
- Groq: https://console.groq.com/keys (бесплатно, Llama 3.1)

## Деплой на Railway
1. https://railway.app → New Project → GitHub
2. Создай свой GitHub-репо, загрузи файлы
3. Добавь env: TOKEN, GROQ_API_KEY
4. Deploy!

Тест: В группе @Bot нет интернета → Ответ из базы.
